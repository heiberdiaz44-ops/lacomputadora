LA COMPUTADORA

Componentes principales del hardware

  • Unidad Central de Procesamiento (CPU): Es el "cerebro" que interpreta y ejecuta las instrucciones. Contiene la Unidad de Control y la Unidad Aritmético-Lógica.

  • Memoria Principal (RAM): Memoria volátil que almacena temporalmente datos y programas en uso.

  • Memoria Secundaria: Dispositivos como discos duros, SSD o unidades ópticas que almacenan datos permanentemente.

  • Dispositivos de Entrada: Permiten ingresar datos (teclado, ratón, micrófono, scanner).

  • Dispositivos de Salida: Muestran o transmiten información procesada (monitor, impresora, altavoces).

  • Placa Base (Motherboard): Circuito que conecta todos los componentes y permite la comunicación entre ellos.

  • Unidad de Alimentación: Suministra energía eléctrica a todos los componentes.

Software y sistema operativo

El software son los programas y datos que la computadora utiliza para funcionar. El sistema operativo (como Windows, macOS o Linux) es el software base que administra el hardware, facilita la ejecución de programas y permite la interacción con el usuario.

Ciclo básico de funcionamiento (Ciclo de instrucción)

La computadora ejecuta tareas siguiendo este ciclo:

  1. Fetch (Captura): La CPU toma la siguiente instrucción desde la memoria principal.

  2. Decode (Decodificación): La unidad de control interpreta la instrucción para saber qué hacer.

  3. Execute (Ejecución): La CPU realiza la operación, puede ser un cálculo, mover datos, etc.

  4. Store (Almacenamiento): Guarda el resultado en la memoria o en un registro.

  5. El ciclo se repite para la siguiente instrucción.

Flujo de datos en una computadora

De manera simplificada, el proceso completo sigue este flujo:

  • El usuario introduce datos usando dispositivos de entrada.

  • Los datos se envían a la memoria principal.

  • La CPU lee instrucciones y datos desde la memoria.

  • La CPU procesa los datos según las instrucciones.

  • Los resultados se almacenan en la memoria o en dispositivos de almacenamiento.

  • Los resultados se envían a dispositivos de salida para mostrarlos al usuario.

Tipos de memoria en la computadora

  • Memoria RAM: Volátil y rápida, guarda datos temporales.

  • Memoria ROM: No volátil, contiene instrucciones básicas para iniciar la computadora (BIOS/UEFI).

  • Cache: Memoria muy rápida dentro de la CPU para acelerar procesos.

  • Memoria secundaria: Almacenamiento permanente (discos duros, SSD).
